# Инструкция по Git

## Основные команды

* git init 
* git commit 
* git status
* git add
* git add.
* git add --all

## Работа с ветками
### Основные команды

* git branch
* git branch branch_name
* git checkout branch name
* git log --graph
* git branch -d branch name
* git branch -D branch name
